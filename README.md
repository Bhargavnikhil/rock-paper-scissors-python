## 🎮 Gameplay

The player selects one of the following:

```
1. Rock
2. Paper
3. Scissors
```

The computer randomly selects one option.

The winner is decided using the following rules:

- Rock defeats Scissors
- Paper defeats Rock
- Scissors defeats Paper
- Same choice results in a Tie
